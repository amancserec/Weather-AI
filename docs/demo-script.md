# Demo Video Script

## Goal

Record a 90 to 120 second demo that clearly shows live data, memory, and practical value.

## Suggested flow

### Scene 1: intro

Say:

> Weather AI is a browser-based weather agent that combines AccuWeather, Hindsight Cloud memory, and Groq for fast personalized forecasting.

Show:

- Landing page hero
- Control panel
- Runtime integration status

### Scene 2: live weather question

Ask:

> Will I need an umbrella tomorrow morning in New Delhi?

Say:

> The agent fetches fresh weather data from AccuWeather before answering.

Show:

- Question submission
- Response mentioning forecast evidence

### Scene 3: memory moment

Set the memory hint:

> The user bikes to work and prefers practical commute advice.

Ask:

> Give me commute advice for tomorrow morning.

Say:

> Hindsight stores user context so the assistant becomes more useful over time.

Show:

- Personalized reply referencing commute needs

### Scene 4: impact close

Say:

> This can help commuters, travelers, field teams, and students make safer daily decisions with real-time weather awareness.

Show:

- Final response
- Launch checklist or architecture slide

## Recording tips

- Keep browser zoom at 110% to improve readability
- Record with OBS, Loom, or Clipchamp
- Keep one backup clip in case live APIs are slow
